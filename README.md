# Useful Commands

1. run local server 
`php -S 127.0.0.1:8000 -t public`
or 
`symfony serve`
2. dump configs: `php bin/console config:dump twig`
3. cache clear: `php bin/console cache:clear`
4. run migration `php bin/console doctrine:migrations:migrate`
